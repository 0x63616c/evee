# Evee MVP Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** A Discord bot (evee) that creates threads from messages and replies using Claude via claude-max-api-proxy.

**Architecture:** Python Discord bot using discord.py. Listens for messages in all channels of a private server. Creates a thread (named with timestamp) from each new channel message, fetches thread history for follow-ups, sends conversation to Claude via the local proxy (OpenAI-compatible API on localhost:3456), and replies in the thread. System prompt loaded from `prompts/SYSTEM.md`.

**Tech Stack:** Python 3.12+, uv, discord.py, openai (Python client), claude-max-api-proxy (Node.js)

### Task 3: Write the bot

**Files:**
- Create: `evee.py`

**Step 1: Write evee.py**

```python
import os
import datetime

import discord
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

# Discord setup
intents = discord.Intents.default()
intents.message_content = True
client = discord.Client(intents=intents)

# LLM client pointing at local proxy
llm = OpenAI(
    base_url="http://localhost:3456/v1",
    api_key="not-needed",
)

MODEL = "claude-sonnet-4"


def load_system_prompt() -> str:
    path = os.path.join(os.path.dirname(__file__), "prompts", "SYSTEM.md")
    with open(path) as f:
        return f.read().strip()


async def get_thread_history(thread: discord.Thread) -> list[dict]:
    """Fetch all messages in a thread and format as OpenAI messages."""
    messages = []
    async for msg in thread.history(limit=100, oldest_first=True):
        if msg.author == client.user:
            messages.append({"role": "assistant", "content": msg.content})
        else:
            messages.append({"role": "user", "content": msg.content})
    return messages


async def ask_llm(conversation: list[dict]) -> str:
    """Send conversation to LLM and return response."""
    system_prompt = load_system_prompt()
    messages = [{"role": "system", "content": system_prompt}] + conversation
    response = llm.chat.completions.create(
        model=MODEL,
        messages=messages,
    )
    return response.choices[0].message.content


async def split_and_send(channel, text: str):
    """Send a message, splitting into chunks if over 2000 chars."""
    while text:
        if len(text) <= 2000:
            await channel.send(text)
            break
        # Find a good split point
        split_at = text.rfind("\n", 0, 2000)
        if split_at == -1:
            split_at = 2000
        await channel.send(text[:split_at])
        text = text[split_at:].lstrip("\n")


@client.event
async def on_ready():
    print(f"evee is online as {client.user}")


@client.event
async def on_message(message: discord.Message):
    # Ignore own messages
    if message.author == client.user:
        return

    # Message in a thread — continue conversation
    if isinstance(message.channel, discord.Thread):
        thread = message.channel
        conversation = await get_thread_history(thread)
        response = await ask_llm(conversation)
        await split_and_send(thread, response)
        return

    # Message in a channel — create a new thread
    now = datetime.datetime.now().strftime("%b %-d - %-I:%M%p").lower()
    thread = await message.create_thread(name=now)
    conversation = [{"role": "user", "content": message.content}]
    response = await ask_llm(conversation)
    await split_and_send(thread, response)


client.run(os.getenv("DISCORD_BOT_TOKEN"))
```

### Task 4: Discord Bot Setup Instructions (for Cal)

These are instructions for the user, not code tasks.

**Step 1: Go to the Discord Developer Portal**

https://discord.com/developers/applications

**Step 2: Click "New Application", name it "evee"**

**Step 3: Go to the "Bot" tab**

- Click "Reset Token" and copy it — you'll need this for `.env`
- Under "Privileged Gateway Intents", enable:
  - **Message Content Intent** (required for reading message content)

**Step 4: Go to the "OAuth2" tab**

- Under "OAuth2 URL Generator":
  - Scopes: select `bot`
  - Bot Permissions: select `Send Messages`, `Create Public Threads`, `Send Messages in Threads`, `Read Message History`
- Copy the generated URL and open it in your browser to add evee to your server

**Step 5: Create `.env` in the project root**

```
DISCORD_BOT_TOKEN=your-token-from-step-3
```
